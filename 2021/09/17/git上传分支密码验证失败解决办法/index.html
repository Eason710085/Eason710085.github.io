Created2021-09-17|Updated2022-02-12|技术

GitHub 在对 Git 操作进行身份验证时不再接受帐户密码了，
因此需要添加 PAT (Personal Access Token)

在Mac系统中：

  1. 进入 Keychain Access
  2. 搜索Github
  3. 找到 github.com –>Intertent Account
  4. 勾选显示密码按钮
  5. 复制密钥
