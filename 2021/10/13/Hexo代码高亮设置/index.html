Hexo代码高亮设置

Created2021-10-13|Updated2022-02-05|技术
|Post View:

在全局配置文件 _config.yml 中，配置：

highlight:
  enable: true
  auto_detect: false
  line_number: true
  tab_replace: '  '
  wrap: true
  hljs: false
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''
